# Craft World Calculator & Companion

Companion app and calculator for Craft World game resources, featuring:
- Real-time pricing from Katana V3 / GeckoTerminal.
- Visual dark/light preferences.
- Desktop browser notifications for completed factory cycles.

## Running Locally

1. Install dependencies from the root:
   ```bash
   npm install
   ```
2. Run development servers (client + server):
   ```bash
   npm run dev
   ```
